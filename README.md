# CreditSmart - React + Vite

*Autores:* Keneth Anderson Dávila Cuesta e Isai Manuel Narvaez Redondo

## Descripción
Aplicación de ejemplo para consultar, simular y solicitar créditos.

## Tecnologías
- React
- Vite
- React Router

## Instalación
1. npm install
2. npm run dev para iniciarla()

## Instrucciones de la app

1. Abril la terminar de VSCODE y escribir npm run para correr iniciar la app.

## Estructura
- src/ : código fuente
- public/: (si aplica)
- package.json

Ejemplo de commits: "Configuración inicial", "Componente CreditCard creado", ...
